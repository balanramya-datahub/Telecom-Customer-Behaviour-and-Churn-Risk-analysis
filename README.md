# Telecom-Customer-Behaviour-and-Churn-Risk-analysis
Telecom Customer Churn Analysis using Python, Pandas, Matplotlib, and Seaborn to identify customer behavior, churn trends, and revenue insights through exploratory data analysis..

## Features
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Revenue Analysis
- Churn Risk Identification
- Data Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Month-to-month customers show higher churn risk.
- Long-term customers generate more revenue.
- Higher monthly charges increase total revenue.
- Long-term contracts improve retention.
